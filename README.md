# حاسبة نقاط المفاضلة - برنامج فرص

موقع صفحة واحدة لحساب **نقاط المفاضلة بشكل تقديري** حسب الجدول الظاهر في القواعد التنظيمية التي زودتني بها.

## التشغيل محليًا
افتح الملف `index.html` مباشرة في المتصفح.

## النشر على GitHub Pages
1. ارفع ملفات المشروع إلى مستودع GitHub (مثلاً على الفرع `main`).
2. من إعدادات المستودع: **Settings → Pages**
3. اختر:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. احفظ، ثم سيظهر لك رابط الموقع.

> ملاحظة: المشروع Static بالكامل (بدون Backend).

## التنبيه
هذه أداة حسابية غير رسمية لحساب نقاط المفاضلة بشكل تقديري بناءً على القواعد التنظيمية المعلنة وما يُدخله المستخدم من بيانات.
